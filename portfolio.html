Napi kalória számláló

PHP, MySQL, Javascript, CSS

Napi kalória számláló, nyílvántartja a user, továbbá az ételek adatait. A DailyCalorie fül alatt meglehet tekinteni a user adatait, és az aznapi elfogyasztott ételeket, továbbá hozzá lehet adni az elfogyasztott ételeket, ami frissíti a napi kalóriát

Az oldalra beregisztrálás során kiszámolja a user alapján megadott adatokat: súly, magasság, napi mozgás stb, és az alapján, hogy a user hízni, fogyni, vagy megtartani akarja súlyát, kiszámolja a napi kalóriát, napi szénhidrátot stb.. A bejelentkezett userek hozzáadhatnak(100g, vagy db alapján), törölhetnek, és kereshetnek az ételek között(ajax segítségével dinamikusan)
DailyCalories: Ételek kiválasztása, majd étel adag megadása után levonja a napi adatokból az étel adatait, és kilistázza az elfogyasztott ételeket dátummal. Session alapján megtarja az adatokat.
